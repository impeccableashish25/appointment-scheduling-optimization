# Healthcare Appointment Scheduling Optimization

## Project Overview
This project analyzes healthcare appointment data to identify scheduling inefficiencies and patient no-show patterns using Power BI.

## Tools Used
- Power BI
- DAX
- Excel

## Dataset
The dataset contains patient appointment records including scheduling date, appointment time, waiting time, and attendance status.

## Key KPIs
- No Show Rate
- Attendance Rate
- Average Waiting Time
- Appointment Distribution by Time Slot

## Key Insights
- Evening appointments had the lowest no-show rate (~6.8%) despite the highest waiting time (~64 minutes).
- Afternoon slots recorded the highest no-show rate (~7.3%).
- Early morning slots had the shortest waiting time (~18 minutes).

## Business Recommendations
- Increase appointment capacity in evening hours.
- Investigate operational delays in afternoon scheduling.
- Implement reminder notifications for early morning appointments.
